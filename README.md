# Ralizm ROFL Telegram Bot

Сатирический Telegram-бот на Python + Gemini, который генерирует посты в стиле вымышленного Тимофея Рализма: канцелярит, договоры, Фонд, юридические отмазки, маты и унитазная политическая метафизика.

## Установка

```bash
python -m venv .venv
```

Windows:

```bash
.venv\Scripts\activate
```

Linux/macOS:

```bash
source .venv/bin/activate
```

Поставить зависимости:

```bash
pip install -r requirements.txt
```

Скопировать env:

```bash
copy .env.example .env
```

На Linux/macOS:

```bash
cp .env.example .env
```

Открой `.env` и вставь ключи:

```env
TELEGRAM_BOT_TOKEN=...
GEMINI_API_KEY=...
```

Запуск:

```bash
python bot.py
```

## Команды

- `/start` — помощь
- `/post тема` — сгенерировать пост
- `/publish тема` — сгенерировать и отправить в канал из `CHANNEL_ID`
- `/style normal` — более сухой официальный стиль
- `/style rofl` — угарный стиль
- `/style toilet` — стиль “всё в унитаз”
- `/style max` — максимально ебанутый сатирический режим
- `/examples` — примеры тем

Можно просто написать боту тему обычным сообщением.

## Пример

```text
/post рабочие очистной станции требуют отменить добровольный взнос
```

## Важно

Это художественная пародия на вымышленного персонажа. Не используй бота для выдачи текста за реальные официальные заявления, угроз, травли реальных людей или призывов к насилию.

Если ты уже где-то засветил настоящий токен Telegram-бота — перевыпусти его через BotFather.
